## 🚀 How to Use This Repo

1. Clone the repository:
   ```bash
   git clone https://github.com/rozeen-baniya/learn.py.git
   cd learn.py
   ```

2. Navigate through folders and explore topics.

3. Run any Python script:
   ```bash
   python filename.py
   ```
